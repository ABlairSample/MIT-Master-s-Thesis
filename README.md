D65.csv is for standard illuminant xyz2rgb conversion. Use Matlab's signal and image processing toolbox.

CIEXYZ61.csv is the data for converting reflectivity spectra to XYZ values

Icosphere.m is a CLUTCH script from Wil O.C. Ward 19/03/2015, University of Nottingham, UK. Read more here: https://www.mathworks.com/matlabcentral/fileexchange/50105-icosphere

imageNorming.m was used to adjust the brightness of ping-pong ball images, normalized based on their exposure time.

scatteringPhotMat_Topcoat_v2_62624.m is the main model script. NMesh = 4 was a standard minimum mesh size for reasonably accurate results.

tcolor.m is a plotting script.
